# Claude Codeが2026年7月に迎えた2つの転換点(Sonnet 5とバックグラウンドサブエージェント)

## リード文

Claude Codeは2026年6月末から7月頭にかけて、開発体験を大きく変える2つの更新を立て続けに出した。デフォルトモデルのSonnet 5への切り替えと、サブエージェントのバックグラウンド実行の標準化だ。

## 本文

**Sonnet 5がデフォルトに**。v2.1.197でClaude Sonnet 5がClaude Codeのデフォルトモデルになった。最大の特徴は1Mトークンのネイティブコンテキストウィンドウで、これはデフォルトかつ上限という位置づけ(小さいコンテキスト版は存在しない)。ただし新しいトークナイザーにより同じテキストでも約30%多くトークンを消費するため、体感できる「実効コンテキスト量」は単純な数字ほど増えていない点には注意が必要だ。価格は2026年8月31日まで$2/$10 per Mtokのプロモーション適用。

**サブエージェントが標準でバックグラウンド実行に**。v2.1.198では、サブエージェントがデフォルトでバックグラウンド実行になった。メインの会話は動き続けたまま、サブエージェントの完了時に通知を受け取る形になり、完了したバックグラウンドエージェントは自動でcommit・push・draft PR作成まで行う。

この2つの変化が同時期に来たことについて、両者の因果関係は検証済み知識の範囲では確認できていない(あくまで筆者の推測になるが、大きなコンテキストウィンドウとバックグラウンド並行実行は相性が良い組み合わせではある)。
